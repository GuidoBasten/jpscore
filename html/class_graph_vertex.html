const GraphEdge * GraphVertex::GetCheapestDestinationByEdges ( const Point &  position) const

this is a modified version of disjkstra. it considers that the edges length depends on the predecessor edge! this is used to calculate the "exact" path length. for this vertices and edges are exchanged in dijkstra algorithm structure. we do not visit vertices, but edges. after that we identify the edge.destination with the corresponding vertex.
